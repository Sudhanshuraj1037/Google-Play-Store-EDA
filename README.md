# 📊 Google Play Store Exploratory Data Analysis

## Overview

This project performs an end-to-end Exploratory Data Analysis (EDA) on the Google Play Store dataset using Python.

The project focuses on identifying trends in app ratings, installs, pricing, reviews, categories, genres, and user engagement.

---

## Dataset

Google Play Store Dataset

---

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

---

## Analysis Performed

- Data Cleaning
- Duplicate Detection
- Missing Value Analysis
- Rating Distribution
- Install Distribution
- Price Analysis
- Category Analysis
- Genre Analysis
- Feature Engineering
- Correlation Analysis

---

## Features Created

- Revenue Potential
- Review Ratio
- Success Index
- App Age

---

## Key Findings

- Most apps are Free.
- Ratings mainly lie between 4.0–4.5.
- Reviews strongly correlate with installs.
- Price has little effect on ratings.
- Only a few apps dominate installs.
